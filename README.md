# 🎬 CineVerse Analytics

A PostgreSQL portfolio project created to practice SQL for Data Analytics using a custom movie industry dataset..

## 📌 Project Goals

- Practice SQL fundamentals
- Learn JOINs and relationships
- Perform data aggregation
- Use CTEs and Subqueries
- Apply Window Functions
- Explore Query Optimization

## 🛠 Technologies

- PostgreSQL
- SQL
- pgAdmin 4
- GitHub
